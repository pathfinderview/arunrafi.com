Plain Text Is the Test

Every AI agent in the world reads .md files. Instructions, memory, skills, prompts. The most powerful software ever built runs on the same format a teenager uses for class notes.

For thirty years software got heavier. Proprietary formats. Custom parsers. IDE-specific project files. PDFs wrapped in DRM. Docs that needed a license to open. Everyone assumed the future would demand more sophistication. It asked for plain text.

Something quiet happens when an idea sits in a .md file. No fonts. No colors. No animation. No deck. Just words, in a line, in front of you. The format gives the idea nowhere else to go. If it holds, it holds in plain text first.

The simplest format turned out to be the honest one. Decoration is not meaning. Polish is not thought. All the chrome stacked on documents over three decades was a tax on clarity, not a contribution to it.

When it came time to instruct the most complex systems ever built, the simplest format ever built turned out to be the right one. The machines picked the format that carries weight without asking the weight to wear a costume.

Ideas that survive unformatted are ideas. The rest were costumes.
