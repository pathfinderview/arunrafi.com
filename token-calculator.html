You send it once. The model may reread it for the rest of the conversation.

In a long chat, old messages stay in context. That means a short prompt is not just cheaper once, it can be cheaper over and over again as the model keeps carrying it forward.

How this works

Cheap text is familiar text.

Language models do not read one character at a time. They read tokens: reusable chunks such as common words, punctuation, number groups, and sub-word fragments.

If a phrase is common, it often fits into fewer tokens. If it contains typos, rare names, URLs, code, or other scripts, it usually breaks into more pieces and becomes more expensive.

This demo is intentionally approximate so the pattern is easy to see on screen. It is built for intuition, not exact billing accuracy.

  • Common words are treated as single tokens in this first version.
  • Punctuation becomes separate tokens, so messy formatting adds overhead.
  • Long unknown strings get chopped into smaller pieces to mimic fragmentation.
